## ⚡ Quick Start

### Prerequisites

```bash
# Gereksinimleri listele
node --version  # >= 18.x
npm --version   # >= 9.x
```

### Installation

```bash
# Repoyu klonla
git clone [REPO_URL]
cd test-deploy-app

# Bağımlılıkları yükle
npm install

# Geliştirme sunucusunu başlat
npm run dev
```

## 🛠️ Development

### Komutlar

| Komut | Açıklama |
|-------|----------|
| `npm run dev` | Geliştirme sunucusunu başlat |
| `npm run build` | Production build oluştur |
| `npm run test` | Testleri çalıştır |
| `npm run lint` | Linting kontrolleri |

### Branching Strategy

```
main          ← Production-ready kod
├── develop   ← Geliştirme branch'i
│   ├── feature/[name]   ← Yeni özellikler
│   ├── bugfix/[name]    ← Bug düzeltmeleri
│   └── hotfix/[name]    ← Acil düzeltmeler
```
