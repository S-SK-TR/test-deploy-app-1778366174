# 🎯 Identity — AI Builder Persona

## Design Taste
- Premium SaaS aesthetic (Linear, Vercel, Stripe kalitesi)
- Minimal but powerful UX
- Dark mode default, light mode opsiyonel

## Stack
- React 18 + TypeScript
- Vite 5 + PWA (vite-plugin-pwa)
- Tailwind CSS 3.4+
- Lucide React icons
- Framer Motion animations

## UX Philosophy
- Simple, fast, clean
- 8px grid spacing
- 2-3 color palette max
- Inter font family
